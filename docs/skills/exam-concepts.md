# Exam Concepts Skill

Use this skill when the user is preparing for a class exam, certification, or
written test that emphasizes definitions and applied distinctions.

Pair it with `study-card-maintainer.md`. For Markdown review notes, also use
`markdown-to-deck.md`.

## Output

- Cards for definitions, comparisons, formulas, and common traps.
- Scenario cards that ask for the best choice or next step.
- Text entries for short passages, case studies, or summaries from source
  material.
- Notes that group related ideas by concept rather than by page order.

## Agent Checklist

1. Identify the exam, course, or topic scope from the source material.
2. Extract one testable idea per card.
3. Create comparison cards when two terms are easy to confuse.
4. Create scenario cards only when the source supports the expected answer.
5. Put formulas, exceptions, and traps into separate cards.
6. Mark unsupported or conflicting source material as `needs-review`.

## Rules

- Do not turn unsupported guesses into answer keys.
- Keep each card focused on one testable idea.
- Preserve source references so the user can audit a card later.
- Mark ambiguous or conflicting source material for review.

## Useful Tags

- `definition`
- `comparison`
- `formula`
- `scenario`
- `trap`
- `needs-review`
